# CleanStream API Pricing and Marketing Strategy

## Positioning

CleanStream should be positioned as **the fastest way to turn messy customer data into production-ready records** for SaaS imports, partner feeds, and onboarding pipelines.

## ICP (Ideal Customer Profile)

1. **B2B SaaS companies (Seed to Series C)**
   - Frequent CSV/XLSX customer imports.
   - Small engineering teams with limited data-infra bandwidth.

2. **Marketplace and fintech operations teams**
   - Multi-source partner files with inconsistent formats.
   - High cost of reconciliation and data correction.

3. **Data platform teams at mid-market companies**
   - Need a reusable ingestion/normalization layer with API controls.

## Packaging & Pricing

### Free (Developer)

- 50k records/month
- Core ingest + normalize + validate
- Community support
- Single API key

### Growth ($299/month)

- 2M records/month included
- Overage: $0.20 per additional 10k records
- Priority email support
- 10 API keys
- 30-day audit retention
- Basic dashboard/usage analytics

### Scale ($1,499/month)

- 20M records/month included
- Overage: $0.12 per additional 10k records
- SLA (99.9%), faster support response
- 100 API keys
- 180-day audit retention
- Webhooks + advanced validation controls

### Enterprise (Custom, starts ~$4k/month)

- Dedicated environment / VPC options
- SSO/SAML, RBAC, SCIM
- Custom retention and compliance package
- Volume discounts and annual contract

## Pricing Principles

- **Primary metric:** records processed (clear customer value alignment).
- **Secondary controls:** API key count, retention window, SLA.
- **Land-and-expand:** generous free tier, low-friction growth tier, enterprise governance upsell.

## Go-To-Market Motion

### 1) Developer-led acquisition

- Publish copy-paste quickstarts (TS/Python) and 10-minute "messy CSV to clean JSON" demos.
- Ship migration guides from DIY ETL scripts.
- Create public benchmark pages for accuracy/latency.

### 2) Content and SEO

- Target intent keywords:
  - "CSV normalization API"
  - "schema inference API"
  - "clean imported data"
  - "validate spreadsheet uploads"
- Publish conversion-focused technical posts with runnable examples.

### 3) Product-led conversion

- In-product usage milestones + upgrade prompts.
- Weekly digest: records cleaned, error rate reduced, time saved.
- ROI calculator on pricing page (engineering hours saved/month).

### 4) Sales-assisted expansion

- Outbound to SaaS companies with import-heavy products.
- Partnership channel with integration agencies and data consultancies.

## Messaging

### Core message

"Stop spending engineering time fixing broken imports. CleanStream standardizes, validates, and audits inbound data in one API call."

### Proof points to emphasize

- Multi-format ingestion (CSV/XLSX/XML/JSON)
- Automatic schema inference + normalization
- Idempotent ingest retries
- Audit trail and validation visibility

## 90-Day Execution Plan

### Days 1–30

- Launch pricing page + self-serve signup.
- Ship SDK snippets and sample app.
- Publish 3 technical SEO articles.

### Days 31–60

- Add onboarding email sequence.

- Activation: first successful ingest within 24h.
- PQL rate: teams crossing free-tier monthly threshold.
- Paid conversion rate from activated workspaces.
- Net revenue retention (expansion via volume).
- Logo churn by segment.
